# Third-party notices

Kiebitz distributions bundle third-party software. The corresponding notices,
license text, exact source revision, binary origins, and archive checksums live
in [`src-tauri/resources/stockfish/`](src-tauri/resources/stockfish/).

These files are included as application resources in both the desktop and
Android bundles. The release workflow verifies the documented SHA-256 hashes
before packaging Stockfish.
